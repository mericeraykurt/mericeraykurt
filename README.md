**[🔊 Phonic Bridge](https://github.com/mericeraykurt/phonic-bridge)** — Lossless real-time audio streaming over UDP. Capture audio from any application and stream it to anyone, anywhere. Built with C++17, WASAPI, and raw sockets.

### 🧠 `> whoami`

I've been interested in computers and tech since I was a kid — not because someone told me to, but because I kept finding things that didn't exist and got tired of waiting for someone else to make them. Same story with music — I couldn't find the sounds I craved, so I started producing my own.

I'm a Windows developer, music producer, and a glitch/bug/exploit hunter at heart. I love breaking things apart to understand how they work, finding the one edge case nobody thought of, and then building something better from scratch. If there's a bug hiding somewhere, I'll find it — and if there's a tool missing, I'll build it.

I focus on low-level systems programming — direct hardware access, kernel-mode APIs, real-time audio, and network protocols. No frameworks, no abstractions, just raw performance.

My philosophy: **"If I can still find a bug in it, it doesn't work. If I can't find a bug in it, I'm not looking hard enough."**
